# JSpek

Vanilla JavaScript [Spek](https://www.spek.cc/)  
Work in progress  

[DEMO](https://nicopowa.github.io/jspek)  

[INFO](https://nicopr.fr/jspek)  

## Features

- Client side only
- Multiple instances
- Browse or drop files
- Export view to PNG
- Double-click to remove
- Installable web app

6.5KB JS + 1.2KB CSS
